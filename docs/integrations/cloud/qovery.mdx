---
title: "Qovery"
description: "How to sync secrets from KMS to Qovery"
---

Prerequisites:

- Set up and add envars to [KMS Cloud](https://kms.lux.network)

<Steps>
  <Step title="Authorize KMS for Qovery">
    Obtain a Qovery API Token in Settings > API Token.

    ![integrations qovery api token](../../images/integrations/qovery/integrations-qovery-token.png)

    Navigate to your project's integrations tab in KMS.

    ![integrations](../../images/integrations.png)

    Press on the Qovery tile and input your Qovery API Token to grant KMS access to your Qovery account.

    ![integrations qovery authorization](../../images/integrations/qovery/integrations-qovery-auth.png)

  </Step>
  <Step title="Start integration">
    Select which KMS environment secrets you want to sync to Qovery and press create integration to start syncing secrets.

    ![integrations qovery create](../../images/integrations/qovery/integrations-qovery-create-1.png)

    ![integrations qovery create](../../images/integrations/qovery/integrations-qovery-create-2.png)

    <Note>
      KMS supports syncing secrets to various Qovery scopes including applications, jobs, or containers.
    </Note>

    ![integrations qovery settings](../../images/integrations/qovery/integrations-qovery.png)
  </Step>
</Steps>
