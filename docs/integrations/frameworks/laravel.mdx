title: "Laravel"
description: "How to use KMS to inject environment variables and secrets into a Laravel app."
---

Prerequisites:

- Set up and add envars to [KMS Cloud](https://kms.lux.network)
- [Install the CLI](/cli/overview)

## Initialize KMS for your [Laravel](https://laravel.com/) app

```bash
# navigate to the root of your of your project 
cd /path/to/project

# then initialize KMS
kms init
```

## Start your application as usual but with KMS

```bash
kms run -- <your application start command>

# Example 
kms run -- php artisan serve
```
