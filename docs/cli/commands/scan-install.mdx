```bash
kms scan install --pre-commit-hook
```

## Description
The command `kms scan install` is designed to incorporate various scanning tools seamlessly into your development lifecycle. 
Initially, we are offering users the ability to install a pre-commit hook. This hook conducts an automatic scan for any exposed secrets in your commits before they are pushed.

### Flags
<Accordion title="--pre-commit-hook">
  ```bash 
  kms scan install --pre-commit-hook
  ```
  
  **Description**
  Installs a git pre-commit hook that triggers KMS to scan your staged changes for any exposed secrets prior to pushing.

</Accordion>
